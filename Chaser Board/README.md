# Chaser Board

## Description
This is an LED chaser board circuit that creates running light patterns. The design produces sequential lighting effects where LEDs light up in a pattern, creating a "chasing" visual effect. These boards are commonly used for decorative lighting, indicators, and visual displays.

## Features
- Sequential LED lighting effects
- Creates dynamic chasing patterns
- Suitable for decorative and indicator applications
- Adjustable speed and patterns (depending on implementation)
