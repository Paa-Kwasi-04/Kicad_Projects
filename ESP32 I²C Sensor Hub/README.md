# ESP32 I²C Sensor Hub

## Description
This project is a specialized ESP32-based board designed to serve as a central hub for multiple I²C sensors. The board leverages the ESP32's wireless connectivity to collect data from various I²C sensors and transmit it wirelessly or process it locally. Ideal for distributed sensor networks and IoT monitoring applications.

## Features
- ESP32 microcontroller with Wi-Fi and Bluetooth
- Multiple I²C sensor connections
- Acts as a sensor data aggregation hub
- Wireless data transmission capability
- Suitable for environmental monitoring and IoT projects
