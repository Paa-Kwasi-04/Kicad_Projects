# ACDC Convertor

## Description
This is an AC-to-DC converter (rectifier) circuit board designed to convert alternating current (AC) to direct current (DC). The board includes rectification, filtering, and voltage regulation components to provide a stable DC output from an AC input source.

## Features
- AC to DC power conversion
- Filtering and voltage regulation
- Suitable for powering DC electronic devices from AC mains
- Compact PCB design
