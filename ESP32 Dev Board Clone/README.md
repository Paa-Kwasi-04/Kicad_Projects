# ESP32 Dev Board Clone

## Description
This is a clone/reproduction of the popular ESP32 development board. The ESP32 is a powerful microcontroller with built-in Wi-Fi and Bluetooth capabilities, making it ideal for IoT projects. This custom PCB replicates the functionality of commercial ESP32 development boards with all necessary supporting circuitry.

## Features
- ESP32 microcontroller with Wi-Fi and Bluetooth
- USB-to-serial interface for programming
- Power regulation circuitry
- GPIO breakout pins
- Reset and boot buttons
- Compatible with ESP32 development ecosystem
