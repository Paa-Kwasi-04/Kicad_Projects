# 8 Channel Relay

## Description
This is an 8-channel relay board design for controlling multiple high-power devices or circuits. Each relay channel can be independently controlled to switch AC or DC loads. This board is commonly used in home automation, industrial control systems, and other applications requiring multiple switched outputs.

## Features
- 8 independent relay channels
- Suitable for controlling multiple loads
- Compatible with microcontroller control (Arduino, ESP32, etc.)
- Can switch both AC and DC loads
